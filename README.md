# 🚀 TaskFlow Manager 2025

TaskFlow Manager is a full-stack **Task Management System** built using the **MERN stack**.  
It supports **JWT authentication**, **role-based dashboards (Admin/User)**, and full **CRUD operations** for tasks with priority and status tracking.

---

- **Frontend (Vercel):**  
  https://task.vercel.app  
  *(use the Production URL from Vercel)*

- **Backend (Render):**

## ✨ Features

### 🔐 Authentication
- User Signup & Login
- JWT-based authentication
- Secure protected routes

### 👥 Role-Based Access
- **Admin Dashboard**
  - View all tasks
  - Create, update, delete tasks
- **User Dashboard**
  - View assigned tasks
  - Manage personal tasks

### 📝 Task Management
- Create tasks with:
  - Title
  - Description
  - Due date
  - Priority (Low / Medium / High / Urgent)
  - Status (To Do / In Progress / Review / Completed)
- Update & delete tasks
- Real-time UI updates

### 🎨 UI
- Clean landing page
- Responsive layout
- Dashboard-style task cards
- Form-based task creation panel
